# TalentFlow – Banco de Talentos

![Laravel](https://img.shields.io/badge/Laravel-13.x-FF2D20?logo=laravel)
![PHP](https://img.shields.io/badge/PHP-8.3-777BB4?logo=php)
![Docker](https://img.shields.io/badge/Docker-✓-2496ED?logo=docker)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-✓-06B6D4?logo=tailwindcss)

Sistema web para cadastro e gestão de currículos, desenvolvido com Laravel e Blade.  
Permite que candidatos enviem seus dados para áreas específicas, enquanto gestores e o RH acompanham as candidaturas com diferentes níveis de acesso.

---

## 🧱 Stack

- **Backend:** Laravel 13 (PHP 8.3)
- **Autenticação:** Laravel Fortify (headless)
- **Frontend:** Blade + Tailwind CSS + Vite
- **Banco de Dados:** MySQL 8
- **E‑mail (testes):** Mailpit
- **Ambiente:** Docker (PHP‑FPM, Nginx, Node)
